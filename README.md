# Ticket Management

#### Author: Ethan Zhang
#### Created At: 2021-03-02

## Build

- runtime environment: Nodejs

```
npm install
npm run build
```

## Run

```
npm run start
```

## Features

1. View searchable fields
2. Search by Users, Organizations and Tickets by fields
3. Supported field types: string, number, boolean, array
